# Instagram User Behavior Analysis (SQL)

This project performs complete data cleaning and analysis using SQL.

## Steps Covered

1. Data exploration
2. Duplicate detection
3. Missing value handling
4. Outlier detection
5. Data standardization
6. Feature engineering
7. Behavioral analysis
8. User segmentation
9. Correlation analysis
10. View creation for BI tools

## Technologies
- MySQL
- SQL Window Functions
- Data Cleaning
- Data Analysis

## Dataset
Instagram user behavior dataset with demographic, engagement, and lifestyle variables.

## Use Cases
- Social media analytics
- Behavioral segmentation
- Engagement analysis
- Data cleaning pipeline
